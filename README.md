# TicTacToe Game in Console

TicTacToe game with traditional rules of the game.

Written in C# using .NET Core 3.1

## Features
- Readable Menu
- Players statistics (paginated or TOP 10 players)
- Saving statistics to file
- Duel statistics

## Libraries
- Newtonsoft.Json
